LEaRN The Lafayette Engagement and Research Network

Dev Dashboard setup

Original CGI Document

Clone repo available at https://alm.cgifederal.com/scm/projects/STORMNET/repos/lafayette-dash/browse;

Pull from feature/develop for Dev changes.

Create new branch for making changes.

Install node.js. https://nodejs.org/en/ (LTS at time of writing this is 6.11.2)

Navigate to clones project directory

Run npm install to download and install dependencies in package.json.

Run npm start to run webpack server. This will open your default browser. As you make changes the server will reload and restart in the browser. This can be customized.

Deploy changes and create PR. When deploying to Dev/Prod server, run "npm run build:prod". This will generate the dist build for deployment.
