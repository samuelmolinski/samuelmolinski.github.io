 API

Miles, Brian (CGI Federal)

Status update:

  • Observation creation updates finished (Still in code review, but done; LEARN-103)
    • No longer need to specific feature of interest ID when creating observations
    • No longer need to specify phenomenon time when creating individual observations
    • Datastream observed area is now properly updated (needed to support mobile sensors)
  • Working on fixing JSON (de)serialization of Observations (LEARN-104)
    • e.g. measurements represented as floating point numbers rather than strings, as required by the SensorThings spec.
    • Also using this as an opportunity to revamp the data model to save substantial storage space
    • Code is written, tests failing, and more tests need to be created. Hope to finish by COB 2017/5/10.
